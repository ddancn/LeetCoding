题目：

给定一个带有头结点 head 的非空单链表，返回链表的中间结点。

如果有两个中间结点，则返回第二个中间结点。

示例1：
>输入：[1,2,3,4,5]  
输出：此列表中的结点 3 (序列化形式：[3,4,5])

示例2：
>输入：[1,2,3,4,5,6]  
输出：此列表中的结点 4 (序列化形式：[4,5,6])

## 解法
极其经典的快慢指针又上线了！

单数时的前进过程是这样的：  
```
1->2->3->4->5->null  
↑↑  
   ↑s ↑f  
      ↑s    ↑f  
```
双数时的前进过程是这样的： 
```
1->2->3->4->5->6->null  
↑↑  
   ↑s ↑f  
      ↑s    ↑f  
         ↑s        ↑f  
```
所以有两种情况表示循环结束：`fast == null`或`fast.next == null`
```kotlin
class Solution {
    fun middleNode(head: ListNode?): ListNode? {
        var slow = head
        var fast = head
        while (fast?.next != null) {
            slow = slow?.next
            fast = fast?.next?.next
        }
        return slow
    }
}
```
